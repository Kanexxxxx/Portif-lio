Exercícios de Lógica de Programação

Este projeto foi desenvolvido como parte da minha jornada de aprendizado em programação, com foco na prática da lógica por meio de pequenos desafios interativos. Utilizando HTML para estruturação, CSS para o design da interface e JavaScript para a lógica funcional, a proposta do site é oferecer um ambiente simples e intuitivo onde é possível exercitar conceitos fundamentais da programação.

Os exercícios abordam situações práticas que ajudam a fixar o raciocínio lógico, o uso de estruturas condicionais, laços de repetição, variáveis e entrada de dados. Cada exercício é apresentado de forma clara, com campos de entrada e botões que, ao serem acionados, executam a lógica necessária e retornam a resposta diretamente na tela, promovendo uma experiência dinâmica e visual para o usuário.

Entre os exercícios disponíveis, estão:

  • Tabuada de Pares: Gera a tabuada completa de um número escolhido, utilizando apenas números pares como multiplicadores.
  • Verificador de Par ou Ímpar: Informa se um número digitado é par ou ímpar, reforçando o uso de operadores lógicos.
  • Calculadora de Média: Calcula a média de três notas e exibe se o aluno foi aprovado ou reprovado, simulando um sistema educacional simples.
  • Soma de Números Pares: Soma automaticamente todos os números pares até o valor informado pelo usuário, utilizando laços de repetição.

Todos os exercícios são executados no próprio navegador, sem necessidade de backend, utilizando apenas tecnologias do front-end. O projeto também possui uma interface responsiva e acessível, pensada para facilitar o uso tanto em computadores quanto em dispositivos móveis.

Este projeto foi essencial para o meu desenvolvimento como estudante de programação, permitindo colocar em prática o que aprendi sobre lógica computacional, estruturas básicas da linguagem JavaScript e boas práticas de organização do código. Além disso, foi uma ótima oportunidade para exercitar a criação de interfaces simples e funcionais com HTML e CSS.

O site tem como objetivo não apenas reforçar os meus conhecimentos, mas também servir como recurso para outras pessoas que estão começando a estudar programação e desejam praticar de maneira prática, visual e divertida.

, incentivando o uso da tecnologia para resolver problemas e explorar ideias criativas.

Urna dos Ladrões — Simulador de Votação em Java

O projeto Urna dos Ladrões é uma aplicação desenvolvida em Java com o objetivo de praticar a lógica de programação, entrada de dados via terminal e controle de fluxo. O sistema simula uma eleição fictícia e humorística, com candidatos de nomes irônicos, criando uma crítica bem-humorada ao cenário político. Através dessa abordagem divertida, o projeto também proporciona uma excelente oportunidade para desenvolver habilidades técnicas com foco em algoritmos e estruturas básicas da linguagem Java.

A aplicação é executada no terminal e permite que o usuário vote em diferentes candidatos, além de registrar votos nulos e brancos. Ao final da votação, o programa exibe um resumo com a quantidade de votos de cada candidato, além de identificar o vencedor ou apontar um possível empate. Todos os dados são armazenados em variáveis locais, utilizando contadores simples para acompanhar os votos.

Durante o desenvolvimento, foram utilizados conceitos fundamentais da programação em Java, como:

  • Uso da classe Scanner para entrada de dados pelo teclado;
  • Estruturas condicionais (if/else e switch) para validação dos votos e exibição dos resultados;
  • Laços de repetição (while) para permitir que vários votos sejam inseridos até o encerramento da votação;
  • Organização do código em métodos para facilitar a leitura e a manutenção.

Além do aspecto técnico, o projeto também desenvolve o pensamento lógico e a capacidade de organizar informações de maneira clara e funcional. É uma excelente prática para quem está começando a programar com Java e deseja explorar a criação de aplicativos simples baseados em texto.

A Urna dos Ladrões é um exemplo de como é possível unir programação e criatividade em projetos educativos e divertidos, transformando conceitos básicos em aplicações práticas que simulam situações reais — com um toque de ironia.

Calculadora de investimento

O Investe Mark é uma plataforma online desenvolvida para auxiliar investidores de todos os níveis a calcular e projetar o crescimento de seus investimentos com precisão e facilidade. Utilizando algoritmos avançados, o site permite que os usuários insiram diferentes variáveis, como valor inicial, aporte mensal, taxa de juros e prazo, para simular cenários financeiros personalizados. Dessa forma, o Investe Mark oferece uma visão clara sobre o potencial retorno de aplicações em diversas modalidades de investimento, promovendo um planejamento financeiro mais consciente e estratégico. Com uma interface intuitiva e recursos interativos, o site torna o processo de tomada de decisão mais acessível, ajudando os usuários a maximizar seus ganhos e alcançar seus objetivos financeiros de forma segura e eficiente.
